# DSC180B Website
Website for DSC180B. Project details can be found at the [DSC180B repo](https://github.com/jonathanlo411/dsc180b/).

## Local Usage
1. Clone the repository.
2. Install packages `npm install`
3. Run local server `npm run dev -- --open`
